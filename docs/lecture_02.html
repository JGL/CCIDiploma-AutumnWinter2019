3. Introduction to JavaScript.

  • It's in your browser (Chrome, Firefox, Safari), right now!
  • It was orginally developed in 1995, in 10 days, by one person!
  • It works pretty much everywhere in the software world. As well as in browsers, Javascript runs many apps and on countless servers in the cloud. Apart from the Stackoverflow one, here is another on the State of JavaScript in 2018.
  • It has hilarious bugs and oddities (or used to).
  • It shares some aspects with Java, but they are completely separate programming languages and differ importantly in design.
  • It's the most popularprogramming language in the world in 2019.
  • The p5.js wiki has a great introduction to JavaScript. Let's take a look together.

5. p5.js web editor, Glitch.com, Visual Studio Code.

  • p5.js web editor. We'll be using this primarily to work with for the first half of this term.
  • Glitch. Glitch is great becase you can add whole Javascript libraries to it, databases and connect to other web sites. But they are a startup, not a foundation like Processing is.
  • Both the p5.js web editor and Glitch allow you to write code on the web. As you write it you can share it with others or publish it.
  • Visual Studio Code text editor (use this for making software on your own computer). You'll have to upload your code to the web manually (using an FTP program like CyberDuck on the Mac or WinSCP on Windows) or using Git with GitHub (more on Git and GitHub next week).
  • VS Code is now the most popular text editor amongst developers.
  • Two great articles on VS Code.
  • Brilliant video (as always) on setting up VS Code by Daniel Shiffman.
