2. Abstraction and Object-oriented programming.

  • Abstraction is a concept that has been applied to many fields, including art and computer science.
  • Abstraction is a key feature of object orientated programming.
  • C++ is an example of an object orientated language. C was one of its predecessors and functional. The ++ means increment, hence C++ being one better than C. (-;
  • There are many different ways of programming, also known as paradigms.
  • Two amusing languages are Brainfuck and Whitespace, there are many other Esoteric languages. (-;

3. Fuzzy logic, Computer Vision and Neural Networks.

  • Fuzzy logic is a logic where results be other numbers between 0 and 1 - rather than one or the other. It's very useful for controlling systems which are analogue, like a washing machine.
  • Computer vision in computer science is the study of making computers able to understand images or video.
  • See this Edge.org article on Hidden Layers for an explanation of how biological research directly influenced the creation of Artifical Neural Networks, a key part of modern Machine Learning. Computerphile video on what Deep Learning is. Another Computerphile video on Convolutional Neural Networks and what makes them SO special.
  • t-SNE is a machine learning technique for collapsing multidimensional data into two or three dimensions - simply put, a great way of visualising a set of images and their relationship to each other.
